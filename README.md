# SolidWorks Practice Models

This repository contains a collection of SolidWorks models created to develop and demonstrate core mechanical design skills.

---

## 📌 Overview

The models in this repository focus on building a strong foundation in CAD design, including part modeling, assemblies, and basic mechanical components.

---

## 🛠️ Skills Demonstrated

- Part modeling using SolidWorks  
- Assembly design and constraints  
- Use of features such as extrude, cut, fillet, and chamfer  
- Understanding of mechanical component structure and fit  

---

## 📂 Contents

- Individual part models  
- Mechanical components  
- Basic assemblies  

---

## 🎯 Purpose

This repository was created to:

- Improve proficiency in SolidWorks  
- Develop mechanical design understanding  
- Support larger projects such as robotics and simulation
